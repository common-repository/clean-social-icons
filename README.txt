Adds clean and elegant social icons to your website

== Description ==

Clean Social Icons was created to help you add social icons to your website in an easy way so you can focus on making great content.

== Social Icons Available ==

* Facebook
* Instagram
* Pinterest
* Twitter
* YouTube
* E-mail
* Behance
* Dribble
* Flickr
* Github
* Linkedin
* Odnoklassniki
* RSS
* Snapchat
* Stumbleupon
* Tumblr
* Vimeo
* VK
* Wordpress
* Xing


== Installation ==

1. Upload `clean-social-icons.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How many icons can I have? =

You can have up to 6 icons in the free version.

= How do I make the icons display on my website? =

You can automatically add the icons to your header or display them anywhere with the shortcode [clean-social-icons].
